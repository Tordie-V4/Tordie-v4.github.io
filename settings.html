Tordie Documentation - Settings

Tordie allows for configuration of default values and settings. These settings are stored in the settings.py file.

General Settings

These settings apply to general shapes and other Tordie specific settings.

Settings

  • DEFAULT_STROKE - The default color of shapes. This is initially set to "black".

  • DEFAULT_RADIUS - The default radius size of points. This is initially set to 3.

  • DEFAULT_THICKNESS - The default thickness of lines. This is initially set to 2.

SVG Settings

These options apply to general shapes and other Tordie specific settings.

Settings

  • DEAFULT_WIDTH - The default width of the SVG file. This is initially set to 800.

  • DEFAULT_HEIGHT - The default height of the SVG file. This is initially set to 800.

  • BACKGROUND - The background color for the SVG file. This is initially set to "white".

  • BOUNDS_FIX - A layer of padding around the outside of the SVG to make the bounds easier to see. This is initially set to 16.

  • DISPLAY_WIDTH - The width of the displayed PNG render of the SVG result file. This is initially set to 1000.

  • DISPLAY_HEIGHT - The height of the displayed PNG render of the SVG result file. This is initially set to 1000.

Mathematical Settings

These settings relate to the mathematical properties of the shapes and SVG result file.

Settings

  • SCALE - The radii of the \(x\) and \(y\) axis as a tuple. For example, SCALE = (3, 5) would display \(x \in [-3, 3]\) and \(y \in [-5, 5]\). This is initially set to (10, 10).

Parametric Settings

These settings apply to parametric curves.

Settings

  • DEFAULT_STEP - Number of steps per parametric. These are equal steps the parameter takes until it travels from \(t_0\) to \(t_n\). They are not the literal distances between the points, so some parts of curves may be smoother than others. The higher this value, the smoother the shape will be but the more processing will be needed when handling them. Lower values may not be smooth but will generally run faster. This is initially set to 200.

Advanced Settings

These settings handle mostly advanced mathematical operations and are not recommended to be changed unless you fully understand their purpose. For certain settings, you might find unstable behaviour if the settings are not set correctly.

Settings

  • POINCARE_ERR - The permitted error radius when approximating properties specific to the PoincareLine class. This is initially set to 0.001.

  • MAN_EPS - A manually set machine number, that is the smallest number considered by the Tordie operations - especially when handling approximations. This is initially set to 0.0001.

  • PARAM_REFLECT_M_DIFF - The error radius to determine the intersection point of the observer to mirror line and a parametric curve. This is performed during parametric reflections, and is usually best with higher values. This property is highly unstable, so slight changes can significantly impact results. This is initially set to 0.2.

  • DIFF_STEP - The limit used when evaluating a derivative at a certain point. This approximates the limit when differentiating by first principles, which allows for more accurate derivatives, regardless of the step size for parametrics. This is initially set to 0.001.

Tordie has a simple checking system to make sure that your inputted values for these specific settings are value for operations. You can do so by importing the init function from init.py. For example: 

from init import init
    init()

You can stop printing the results of the checks by simply inputting the keyword argument short=True into the init function. For example: 

from init import init
    init(short=True)

Generally, do not set any of these settings to be less than MAN_EPS (the manual machine number), otherwise it will raise an error.
